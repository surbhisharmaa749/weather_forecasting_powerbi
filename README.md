This project is an interactive **Weather Forecasting Dashboard** developed using **Power BI** to analyze and visualize real-time and forecasted weather data in a clear and user-friendly format.

The dashboard provides comprehensive insights such as **current temperature, 7-day weather forecast, air quality index (AQI), humidity, wind speed, visibility, pressure, UV index, and chance of rain**. It allows users to explore weather conditions across different cities and days through interactive slicers and visuals.

This project demonstrates **end-to-end Power BI development**, including:
- Extracting data from a weather API
- Cleaning and transforming data using **Power Query**
- Creating calculated measures using **DAX**
- Designing an interactive and visually appealing dashboard with a modern glassmorphism-style UI

The main focus of this project is on **data visualization, analytics, and dashboard design**, making it suitable for showcasing Power BI skills in a professional portfolio.
